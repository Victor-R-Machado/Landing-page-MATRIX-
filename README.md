# Victor Rodrigues Machado — Landing Page (Matrix Hacker Theme)

Repositório da minha **landing page pessoal**
Este projeto foi pensado para representar quem eu sou hoje na área de tecnologia, unindo um visual **hacker / Matrix** com informações claras sobre minha trajetória e interesses.

##  Sobre o projeto

Esta landing page é um site **estático**, desenvolvido com **HTML, CSS e JavaScript puro**, sem frameworks, com o objetivo de:

* Apresentar meu perfil profissional de forma moderna
* Servir como cartão de visitas digital
* Explorar criatividade visual com uma temática hacker
* Ser leve, rápido e fácil de hospedar

O destaque visual fica por conta do **background animado estilo Matrix**, com códigos binários caindo em cascata, remetendo ao universo de tecnologia, sistemas e programação.

---

## Tecnologias utilizadas

* **HTML5** — Estrutura da página
* **CSS3** — Estilização, efeitos visuais e layout responsivo
* **JavaScript (Vanilla)** — Animação do efeito Matrix
